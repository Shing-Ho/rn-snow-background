# rn-snow-background

![example][example-url]

## Installation

```bash
npm i rn-snow-background
```

## Usage

```javascript
import SnowBackground from "rn-snow-background";

<SnowBackground />;
```

## Propertise

| name      | description                 | type   | default                |
| :-------- | :-------------------------- | :----- | :--------------------- |
| colors    | Colors of linear background | array  | ["#1b2735", "#090a0f"] |
| snowCount | Color of the footer         | string | 150                    |
